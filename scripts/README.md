# Scripts

Utility scripts for testing, maintenance, and knowledge base management.

## Scripts

- `ingest_knowledge_base.sh` — Chunk and embed knowledge base files into Supabase
- `reembed_all_chunks.sh` — Re-embed all active knowledge chunks
- `test_webhook_verification.sh` — Test Meta webhook handshake
- `test_send_dm.sh` — Send a test DM via Graph API
- `refresh_meta_token.sh` — Manually refresh Meta long-lived token
- `export_n8n_workflows.sh` — Export all n8n workflows as JSON for version control
